Greedy

Details

A simple CSS grid build with flexbox.

Inspired by grd.

  • Simple: Has only 2 base classes and a handful of modifier classes.
  • Powerful: Contains modifiers that are only applied if the window meets the minimum width requirement.
  • Modern: Uses calc() and flexbox and should work on all modern browsers.

Installation

Download the script and include it in your project.

<link rel="stylesheet" href="css/greedy.css">

Usage

Grid modifier classes

Modifiers for align-items:
  • -top
  • -middle
  • -bottom
  • -stretch
  • -baseline
Modifiers for justify-content:
  • -left
  • -center
  • -right
  • -between
  • -around

Cell modifier classes

General modifiers
  • -fill(fills the remaining width)
Modifiers for min-width < 768px(tiny)
  • -1t(1 / 12 of the width of the container)
  • -2t(2 / 12 of the width)
  • ...
  • -12t(12 / 12 of the width
Modifiers for min-width < 992px(small)
  • -1s
  • -2s
  • ...
  • -12s
Modifiers for min-width < 1200px(medium)
  • -1m
  • -2m
  • ...
  • -12m
Modifiers for min-width > 1199px(large)
  • -1l
  • -2l
  • ...
  • -12l

Examples

#1

Cells will get full width when screen size is under 768px(tiny), and half width when screen size is bigger than 991px(medium). 

                <div class="Grid">
                    <div class="Cell -12t -6m red">
                        a
                    </div>
                    <div class="Cell -12t -6m blue">
                        b
                    </div>
                </div>
a
b

#2

First cell will get 8/12 width when screen size is less than 768px(tiny), 6/12 width when screen size is less than 992px(medium) and 4/12 when screen size is bigger than 1200px(large). The second cell will always fill the remaining space. 

            <div class="Grid">
                <div class="Cell -8t -6s -4l red">
                    a
                </div>
                <div class="Cell -fill blue">
                    b
                </div>
            </div>
a
b
